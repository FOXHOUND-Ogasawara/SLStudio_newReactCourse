問題：時間ごとに表示する挨拶を変えれる機能を実装してください
・6:00～9:59：おはよう
・10:00～13:59：こんにちは
・14:00～24:00、0:00～5:59：こんばんは

■第1段階
1. 時間を定義し、適当な数値で初期化する
2. 条件分岐により表示する文言を変える

//1. 時間を定義する
const time = 6;

//2. 条件分岐により表示する文言を変える
if(time > 6 && time < ???) {
    return (
        <h1>おはよう<h1/>
    )
} else if(time > 10 && time < ???) {
    return (
        <h1>こんにちは<h1/>
    )
} else if(time > 14 && time < 24 || time > 0 && time < ???) {
    return (
        <h1>こんばんは<h1/>
    )
}

状況
・1の実装部分でなんかエラー（実際は警告）が起きる
・9:59など分刻みのところの表し方が分からないため条件文が書けずエラーが発生している
→ここで質問してもOK

■第2段階
1. 時間を表す変数timeをnumber型で定義し、適当な数値で初期化する
2. 条件分岐により表示する文言を変える
2-1．6:00~9:59(10:00未満)は「おはよう」
2-2. 10:00~13:59(14:00未満)は「こんにちは」
2-3. 14:00~24:00または、0:00~5:59(6:00未満)は「こんばんは」

//1. 時間を表す変数timeをnumber型で定義し、適当な数値で初期化する
const time:number = 6;

//2. 条件分岐により表示する文言を変える
//2-1．6:00~9:59(10:00未満)は「おはよう」
if(time > 6 && time < 10) {
    return (
        <h1>おはよう<h1/>
    )
//2-2. 10:00~13:59(14:00未満)は「こんにちは」
} else if(time > 10 && time < 14) {
    return (
        <h1>こんにちは<h1/>
    )
//2-3. 14:00~24:00または、0:00~5:59(6:00未満)は「こんばんは」
} else if(time > 14 && time < 24 || time > 0 && time < 6) {
    return (
        <h1>こんばんは<h1/>
    )
}

状況
・1に関してはエラー解消できた
・2に関してエラーは解消できた
・ただ6のときに何も表示されない
・7は「おはよう」が表示される
・10も表示されない
・・・（以下、類似の状況）
→条件分岐の書き方が間違っていそうなのでテキストを見返してみる＆質問してみる
